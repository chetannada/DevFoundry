## ⭐ Star Rating

An interactive and responsive star rating component built with **React.js**, **JavaScript**, and **TailwindCSS**. This tool allows users to select, hover, and clear ratings with smooth animations—ideal for reviews, feedback forms, or any application requiring user ratings.

- 💻 [Source Code](/src/features/star-rating)
- 🌐 [Live Demo](https://devfoundry.netlify.app/star-rating) ⭐

### 🎯 Challenge Overview

#### 🕒 Estimated Completion Time: 30–45 minutes

#### 🛠️ Task Overview:

Build a star rating component that dynamically updates based on user interaction. The UI should be clean, responsive, and optimized for clarity and usability.

#### 📌 Requirements:

- ⭐ **Star Icons**: Display 5 interactive stars that respond to hover and click.
- 🖱️ **Hover Effect**: Highlight stars on hover for preview before selection.
- ✨ **Rating Selection**: Allow users to set a rating from 1 to 5.
- 🧹 **Clear Functionality**: One-click reset to clear the rating.
- ⚛️ **State Management**: Use React hooks to track rating and hover states.
- 🎨 **Styling**: Modern, minimal layout with TailwindCSS animations.

#### 🔍 Development Focus:

- **Component Modularity** → Separate components for stars, rating display, and clear button.
- **UX Clarity** → Smooth hover animations, clear feedback, and intuitive interactions.
- **Performance** → Efficient rendering logic for responsive updates.
- **Accessibility** → Semantic HTML, keyboard-friendly interactions, and clear focus states.

#### 🌟 Additional Considerations:

- Support half-star ratings for more granular feedback.
- Add labels or tooltips (e.g., "Poor", "Excellent").
- Integrate with backend APIs to store ratings.
- Allow customization of star count or icon style.
